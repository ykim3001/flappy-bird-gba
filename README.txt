This game refers to Flappy Bird game. The main character, flappy bird needs to escape through pipe avoiding the predator, big red bird. When the yellow bird collides with the red bird, game over! When you arrives the pipe safely without encountering the red bird, you win! Also, the yellow bird is only given 50 times he can flap his wings so when flapping number would be 0, you lose as well. Use arrow key to move and press backspace to reset. Enjoy!
